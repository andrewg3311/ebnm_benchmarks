Introduction

This code analyzes the results of the DSC replications. Currently, only the point_normal prior is tested.

The point_normal prior assumes \[ \begin{aligned} \theta_j \stackrel{iid}{\sim} \pi_0 \delta_\mu + (1 - \pi_0)\mathcal{N}\Big(\mu, \frac1a\Big) \\ x_j \stackrel{\perp}{\sim} \mathcal{N}\Big(\theta_j, s_j^2\Big) \end{aligned} \] By setting \(\mu := 0\), we can recover the case where there is a point-mass at 0. By setting \(\pi_0 := 0\), we can recover the case where there is no point-mass.
